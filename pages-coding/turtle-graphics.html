Turtle Graphics

p5.js

Created in p5.JS as part of a midterm assignment in a Creative Coding class at New York University.

The assignment was to create an interesting path for a "turtle" to draw in p5.js. Our professor gave us introductory code and requirements for changes he wanted to see made to his code. My code creates a five-pointed spiral that eventually goes offscreen.
