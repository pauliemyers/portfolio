Text Machine

p5.js

Created in p5.JS as the second assignment in a Creative Coding class at New York University.
NOTE: This project creates sound at high frequencies. Please lower your volume.
The assignment was to create a "text machine". Using this script, one can take any text file, strip it of punctuation and capitalization, and pull one word out. A major scale then plays that depends upon the first letter of the word randomly selected. The particular script used in this assignment was a poorly-created version of the Bee Movie Script in which most "c" letters at the starts of words were recorded as "o"'s, so some of the selected words will be incorrectly spelled.
