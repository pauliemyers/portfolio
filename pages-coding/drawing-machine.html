Drawing Machine

p5.js

Created in p5.JS as the first assignment in a Creative Coding class at New York University.



The assignment was to create a "drawing machine". Using this script, one can draw random colors by holding down the left mouse button. With various key inputs, the user can clear the screen, control the color of the drawing, or reduce the size of the ever-growing brush.
